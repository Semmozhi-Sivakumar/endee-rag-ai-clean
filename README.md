# 🤖 Endee RAG AI – Resume Semantic Search System

An AI-powered Resume Analyzer built using **Retrieval Augmented Generation (RAG)** and **Endee Vector Database**, enabling intelligent semantic search, resume ranking, and AI-based candidate matching.

---

## 🚀 Project Overview

Traditional resume filtering relies on keyword matching, which often misses context and meaning.

This project solves that using **semantic search + vector embeddings**, allowing the system to understand the *meaning* of resumes and job descriptions.

It enables:
- Intelligent resume matching
- Semantic search over resumes
- AI-powered candidate ranking
- Context-aware retrieval using RAG pipeline

---

## 🧠 Key Features

- 🔍 Semantic Resume Search (not keyword-based)
- 🤖 RAG-based AI retrieval system
- 📊 Resume ranking using vector similarity
- 📁 Multi-resume support
- ⚡ Fast retrieval using Endee vector database
- 🧾 Job description based candidate matching
